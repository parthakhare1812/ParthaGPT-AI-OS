Handwritten Digit Recognition using Convolutional Neural Networks (CNN)

📌 Project Description:
This project demonstrates a simple yet powerful implementation of handwritten digit recognition using a Convolutional Neural Network (CNN) model trained on the MNIST dataset. It provides an interactive interface in Google Colab where users can upload a custom image of a digit (drawn on paper, MS Paint, or any tool), and the trained model will predict and display the digit.

🎯 Objective:
To build a robust machine learning model capable of accurately identifying handwritten digits (0–9) from images, and to provide an interactive way to test the model with user-uploaded images.

🧠 Technologies & Tools Used:
Google Colab (for development and execution)

Python 3

TensorFlow / Keras (for building and training the CNN)

MNIST dataset (for training and evaluation)

NumPy & Matplotlib (for data processing and visualization)

PIL (Pillow) (for image handling)

Google Colab files module (for image uploads)

🔍 Project Workflow:
1. Data Loading & Preprocessing
Load the MNIST dataset (60,000 training and 10,000 test images of digits 0–9).

Normalize the pixel values and reshape for CNN input format.

2. Model Architecture
A CNN with the following structure:

Conv2D → MaxPooling2D → Conv2D → MaxPooling2D

Flatten → Dense → Dense (Output)

Activation Functions: ReLU and Softmax.

3. Model Training
The model is trained for 5 epochs with Adam optimizer and categorical crossentropy loss.

Validation is performed using the test set.

4. Image Upload & Prediction
User uploads an image (preferably 28x28 grayscale).

Image is preprocessed, reshaped, and normalized.

Model predicts the digit, and the result is displayed alongside the uploaded image.
